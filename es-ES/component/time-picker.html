TimePicker #

Use el Time Picker para input de tipo time.

TIP

This component requires the <client-only></client-only> wrap when used in SSR (eg: Nuxt) and SSG (eg: VitePress).

Selector de tiempo arbitrario #

Can pick an arbitrary time.

Limitar el rango de tiempo #

You can also limit the time range.

Rango de tiempo arbitrario #

Can pick an arbitrary time range.
