Migration #

Esta guía le ayudará a pasar del Element 2.x al Element Plus.

Versión de migración de Vue 3 #

Puede encontrar algunos problemas al usar Element Plus con la compilación de migración Vue 3. Algunos de los componentes dependen de las API internas de Vue 3. Vale la pena intentar ajustar el modo compatConfig a 3, ya sea globalmente o por componente en tu proyecto.

Herramienta de Migración 🛠️ #

Hemos creado una herramienta de migración para que pueda migrar su proyecto desde la interfaz de Element UI a Element Plus. Puedes encontrar la herramienta de migración de código gogo aquí.

Hemos probado esto en Vue Element Admin (Vue2 + Element UI). Puedes encontrar el código transpilado [aquí](https://github. com/gogocodeio/vue-element-admin).
