クイックスタート #

このセクションでは、プロジェクトで Element Plus を使用する方法について説明します。

使用法 #

フルインポート #

バンドルサイズをあまり気にしない場合は、フルインポートを使用する方が便利です。

import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')

オンデマンド インポート #

使用したコンポーネントをインポートするには追加のプラグインを使用する必要があります。

自動インポート Recommend #

まず、 unplugin-vue-components をインストールする必要があります。

npm install unplugin-vue-components

次に、 Vite または webpack の設定ファイルに以下のコードを追加してください。

Vite #
// vite.config.ts
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default {
  plugins: [
    // ...
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
}
Webpack #
// webpack.config.js
const Components = require('unplugin-vue-components/webpack')
const { ElementPlusResolver } = require('unplugin-vue-components/resolvers')

module.exports = {
  // ...
  plugins: [
    Components({
      resolvers: [ElementPlusResolver()],
    }),
  ],
}

他のバンドラ(Rollup, Vue CLI)と設定については、 unplugin-vue-components を参照してください。

手動でインポート #

Element Plusは、ESモジュールに基づいて ツリーシェイク の機能を提供します。

しかし、スタイルのインポートには unplugin-element-plus をインストールする必要があります。 設定方法は ドキュメント を参照してください。

App.vue

<template>
  <el-button>I am ElButton</el-button>
</template>
<script>
  import { ElButton } from 'element-plus'
  export default {
    components: { ElButton },
  }
</script>

// vite.config.ts
import ElementPlus from 'unplugin-element-plus/vite'

export default {
  plugins: [ElementPlus()],
}

スターターテンプレート #

Vue CLI #

プラグイン Element Plus VueCLI プラグイン を用意しました。 vue-cliの場合、Element Plusに基づいて プロジェクトを簡単にセットアップできます。

スターターキットの使い方 #

一般的な プロジェクトテンプレート、 また、 Vite テンプレート を提供します。 そのまま使える汎用的なプロジェクトテンプレートを提供し、さらには Viteテンプレートを使用しています。 のために Laravel をお使いの方は、対応するテンプレートも用意していますので、そのままダウンロードしてお使いいただけます。

グローバル設定 #

When registering Element Plus, you can pass a global config object with size and zIndex to set the default size for form components, and zIndex for popup components, the default value for zIndex is 2000.

ElementPlus を完全紹介。

import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import App from './App.vue'

const app = createApp(App)
app.use(ElementPlus, { size: 'small', zIndex: 3000 })

オンデマンド:

import { createApp } from 'vue'
import { ElButton } from 'element-plus'
import App from './App.vue'

const app = createApp(App)
app.config.globalProperties.$ELEMENT = {
  // options
}
app.use(ElButton)

Nuxt.js を使う #

上記の設定により、プロジェクト内でsizeプロパティを持つすべてのコンポーネントのデフォルトサイズは「small」となり、ポップアップボックスの初期 z-index は 3000 となります。
