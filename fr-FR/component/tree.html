Tree #

Affiche un ensemble de données possédant plusieurs niveaux de hiérarchie ou d'imbrication.

Utilisation de base #

Voici la structure basique.

Noeud-feuille personnalisés en mode lazy #

Checkbox désactivées #

Les checkbox des noeuds peuvent être désactivées individuellement.

Ouverture et sélection par défaut #

Certains noeuds peuvent être ouverts et/ou sélectionnés par défaut.

Sélectionner des noeuds #

Contenu personnalisé #

Le contenu des noeuds peut être personnalisé, afin de pouvoir ajouter des icônes ou des boutons par exemple.

Filtrage #

Les noeuds peuvent être filtrés par mot-clé.

Accordéon #

Vous pouvez utiliser un mode accordéon afin que seul un noeud par niveau soit ouvert.

Noeuds déplaçables #

Vous pouvez déplacer les noeuds par drag'n drop en ajoutant l'attribut draggable.

Méthode #

Tree possède les méthodes suivantes, qui retourne la sélection de noeuds actuelle.

MéthodeDescriptionParamètre
filtrerLe noeud ouvert, la propriété node de TreeNode, TreeNode lui-même.Accept a parameter which will be used as first parameter for filter-node-method
updateKeyChildrenSupprime un noeud, ne marche que si node-key est assigné.(key, data) Accepte deux paramètres: 1. clé du noeud 2. nouvelles données.
getCheckedNodesSi la sélection se fait aussi en cliquant sur le noeud. Si false, la sélection ne se fera qu'en cliquant sur la checkbox.(leafOnly, includeHalfChecked) Accepte deux booléen: 1. default value is false. Si true, retourne seulement un tableau des sous-noeuds sélectionnés. 2. default value is false. If the parameter is true, the return value contains halfchecked nodes
setCheckedNodesLe noeud courant, la propriété node de TreeNodean array of nodes to be checked
getCheckedKeysIf the node can be selected (show-checkbox is true), it returns the currently selected array of node's keys(leafOnly) Booléen, défaut à false. Si true, retourne seulement un tableau des sous-noeuds sélectionnés.
setCheckedKeysLe noeud courant, la propriété node de TreeNode(keys, leafOnly) Accepte deux paramètres: 1. un tableau de clés de noeuds à sélectionner 2. un booléen avec pour défaut false. Si true, retourne seulement un tableau des sous-noeuds sélectionnés.
setCheckedLe noeud cliqué, la propriété node de TreeNode, TreeNode lui-même.(key/data, checked, deep) Accepte trois paramètres: 1. La clé ou les données du noeud 2. un booléen indiquant si sélectionné. 3. un booléen indiquant si profond.
getHalfCheckedNodesSi true, retourne la moitié des noeuds sélectionnés.-
getHalfCheckedKeysIf the node can be selected (show-checkbox is true), it returns the currently half selected array of node's keys-
getCurrentKeyreturn the highlight node's key (null if no node is highlighted)—
getCurrentNodereturn the highlight node's data (null if no node is highlighted)—
setCurrentKeyset highlighted node by key, only works when node-key is assigned(node, shouldAutoExpandParent=true) 1. le noeud. 2. whether to automatically expand parent node
setCurrentNodeset highlighted node, only works when node-key is assigned(key, shouldAutoExpandParent=true) 1. the node to be highlighted 2. whether to automatically expand parent node
getNode(data) la clé ou les données du noeud.(data) le noeud ou ses données à supprimer.
removeremove a node, only works when node-key is assigned(data) the node's data or node to be deleted
appendappend a child node to a given node in the tree(data, parentNode) 1. les données du noeud à ajouter 2. les données du parent, clé ou données.
insertBeforeInsère un noeud avant un autre noeud.(data, refNode) 1. Les données du noeud à insérer 2. Clé ou noeud de référence.
insertAfterInsère un noeud après un autre noeud.(data, refNode) 1. Les données du noeud à insérer 2. Clé ou noeud de référence.

Évènements #

MéthodeDescriptionParamètres
node-clickFiltre les noeuds.Accepte un paramètre qui sera le premier paramètre de filter-node-method.
node-contextmenuAjoute de nouvelles données au noeud, ne marche que lorsque node-key est assigné.L'évènement, le noeud cliqué, la propriété node de TreeNode, TreeNode lui-même.
check-changeSi le noeud peut-être sélectionné (show-checkbox est true), il retourne un tableau des noeuds sélectionnés.three parameters: node object corresponding to the node whose selected state is changed, whether the node is selected, whether node's subtree has selected nodes
checkDétermine quels noeuds sont sélectionnés, ne marche que si node_key est assigné.Un tableau de noeuds qui doivent être sélectionnés.
current-changeSi le noeud peut être sélectionné (show-checkbox est true), retourne un tableau de clés des noeuds sélectionnés.Le noeud fermé, la propriété node de TreeNode, TreeNode lui-même.
node-expandDétermine quels noeuds sont sélectionnés, ne marche que si node_key est assigné.Le noeud modifié, l'objet statut de l'arbre avec quatre propriétés: checkedNodes, checkedKeys, halfCheckedNodes, halfCheckedKeys.
node-collapseDétermine si un noeud est sélectionnable, ne marche que si node_key est assigné.three parameters: node object corresponding to the node closed, node property of TreeNode, TreeNode itself
node-drag-startSi le noeud peut être sélectionné (show-checkbox est true), retourne la moitié des noeuds sélectionnés.two parameters: node object corresponding to the dragging node, event.
node-drag-enterSi le noeud peut être sélectionné (show-checkbox est true), retourne les clés de la moitié des noeuds sélectionnés.three parameters: node object corresponding to the dragging node, node object corresponding to the entering node, event.
node-drag-leaveretourne la clé du noeud actuellement en valeur (null si aucun noeud n'est en valeur).three parameters: node object corresponding to the dragging node, node object corresponding to the leaving node, event.
node-drag-overretourne les données du noeud actuellement en valeur (null si aucun noeud n'est en valeur).three parameters: node object corresponding to the dragging node, node object corresponding to the dragging over node, event.
node-drag-endMet un noeud en valeur par sa clé, ne marche que si node_key est assigné.Le noeud déplacé, le noeud d'arrivée (peut-être undefined), le type de placement (before / after / inner), l'évènement.
node-dropMet un noeud en valeur, ne marche que si node_key est assigné.four parameters: node object corresponding to the dragging node, node object corresponding to the dropped node, node drop type (before / after / inner), event.
