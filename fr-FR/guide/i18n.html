Internationalisation #

Element Plus components are using English by default, if you wish you use other languages, you can get you answer by keep reading.

Configuration globale #

Element Plus provides global configurations

importer ElementPlus depuis 'element-plus'
importer zhCn depuis 'element-plus/es/locale/lang/zh-cn'

app.use(ElementPlus, {
  locale: zhCn,
})

ConfigProvider #

Element Plus also provides a Vue component ConfigProvider for globally configuring locale and other settings.

<template>
  <el-config-provider :locale="locale">
    <app />
  </el-config-provider>
</template>

<script>
  import { defineComponent } de 'vue'
  import { ElConfigProvider } from 'element-plus'

  import zhCn from 'element-plus/lib/locale/lang/zh-cn'

  export default defineComponent({
    composantes: {
      ElConfigProvider,
    },
    setup() {
      return {
        locale: zhCn,
      }
    },
  })
</script>

Utilisation du CDN #

If you are using Element Plus via CDN, then you need to do this, let's again take unpkg as an example

<script src="//unpkg.com/element-plus/dist/locale/zh-cn">
  <script>
  app.use(ElementPlus, {
    locale: ElementPlus.lang.zhCn
  })
</script>

Questions fréquemment posées #

Si je veux remplacer le pack de langue par défaut pour réduire la taille, comment faire? #

Lorsque la langue par défaut de votre application n'est pas Anglais, vous allez avoir besoin de pour importer un autre fichier de langue, qui augmentera la taille du bundle puisque vous avez à la fois English et la langue que vous souhaitez regroupée, vous pouvez utiliser le plugin NormalModuleReplacementPlugin fourni par webpack pour remplacer le fichier de langue par défaut, pour que vous ne receviez que 1 fichier de langue. Ajoute le code ci-dessous dans ton webpack.config.js pour le faire fonctionner.

webpack.config.js

{
  plugins: [
    new webpack.NormalModuleReplacementPlugin(
      /element-plus[\/\\]lib[\/\\]locale[\/\\]lang[\/\\]lang[\/\\]fr/,
      'element-plus/lib/locale/lang/zh-cn'
    ),
  ]
}
