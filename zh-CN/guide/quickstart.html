快速开始 #

本节介绍如何在您的项目中使用 Element Plus。

用法 #

完整导入 #

如果您对捆包的大小不很关心，使用完全导入更方便。

import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')

即时导入 #

您需要使用附加插件来导入您使用的组件。

自动导入 推荐中 #

首先你需要安装 unplugin-vue-components。

npm 安装 unplugin-vue-components

然后将下面的代码添加到您的 Vite 或 webpack 配置文件。

Vite #
// vite.config.ts
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default {
  plugins: [
    // ...
    组件(
      解析器：[ElementPlusResolver()]，
    })，
  ]，
}
Webpack #
// webpack.config.js
const components = require('unplugin-vue-components/webpack')
const { ElementPlusResolver } = require('unplugin-vue-components/resolvers')

module.exports =
  // ...
  插件：[
    Components(volume
      resolvers: [ElementPlusResolver()],
    }),
  ],
}

欲了解更多的捆绑程序(Rollup, Vue CLI) 并请参考 unplugin-vue-components

手动导入 #

元素加成出框 基于ES模块的 Ture Shaking 功能。

但是您需要安装 unplugin-element-plus 来进行样式导入。 如何配置它，请参阅 docs。

App.vue

<template>
  <el-button>I am ElButton</el-button>
</template>
<script>
  import { ElButton } from 'element-plus'
  export default {
    components: { ElButton },
  }
</script>

// vite.config.ts
从 'unplugin-element-plus/vite' 导入ElementPlus

Export default &
  plugins: [ElementPlus()],
}

启动模板 #

Vue CLI #

我们准备了一个插件 Element Plus VueCLI 插件。 对于 vue-cli, 你可以轻松地在 Element Plus 上设置一个基于 的项目。

使用 Starter Kit #

我们提供了通用的项目模板，你可以直接使用，另外我们还提供了 Vite 模板。 对于 Laravel 用户，我们也准备了相应的模板，同样可以直接下载使用。

全局配置 #

当注册元素加速度时， 您可以通过一个带有 大小 和 的全局配置对象zIndex 来设置默认值 大小 表单组件. 和 zIndex 用于 弹出组件，默认值为 zIndex 为 2000。

完整引入 ElementPlus：

从 'vue' 导入 { createApp }
从 'element-plus'
从 './App.vue' 导入应用程序

const app = createApp(App)
app.use(ElementPlus, { size: 'small', zIndex: 3000 })
