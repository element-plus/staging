翻译 #

文档 #

在本章中，我们将讨论如何帮助翻译 Element Plus 开发文档

背景 #

在我们升级文档架构之前，每个文档的更新都需要 5 种语言, 大多数贡献者使用非本地语言的在线翻译， 有时这不仅是不准确的，而且也会对文档阅读者造成理解压力。

因此，我们决定升级在线文档网站。

  • 从 webpack 到 Vite
  • 从手动维护到自动化更新

我们通过对接 Crowdin 网站, 实现了文档网站自动化的第一步

如何翻译文档？ #

  1. 在 Crowdin 网站创建账号, 推荐使用你的 GitHub 账号来授权 Crowdin 自动登录.
  2. 进入 Element Plus 项目
  3. 选择你想要翻译的语言
  4. 找到想要翻译的文件
  5. 开始翻译

这个过程很简单，crowin 的用户界面也非常直观，还可以选择中文模式, 你应该很容易就可以学会如何使用它。 在提交你的译文后，一旦经校对官批准，它将自动部署到线上网站。

如何成为校对官？ #

你可以通过 发起 issue 的方式在 Crowdin 通知我们, issue 内请告知你希望成为哪个语言的校对官
