Avatar 头像 #

头像可以用来代表人物或对象， 支持三种类型：图标、图片和字符.

基础用法 #

通过 shape 和 size 设置头像的形状和大小。

不同状态 #

支持三种类型：图标、图片和字符

图片加载失败的 fallback 行为 #

当图片加载发生错误时加在备用值

图片如何适应容器框 #

当使用图片作为用户头像时，设置该图片如何在容器中展示。与 object-fit 属性一致

Attributes #

参数说明类型可选值默认值
图标集合设置头像的图标类型，参考 Icon 组件String
size设置头像的大小number/stringnumber / large / medium / smalllarge
shape设置头像的形状Stringcircle / squarecircle
src图片头像的资源地址string
srcSet以逗号分隔的一个或多个字符串列表表明一系列用户代理使用的可能的图像String
alt描述图像的替换文本string
fit当展示类型为图片的时候，设置图片如何适应容器框stringfill / contain / cover / none / scale-downcover

Events #

事件名说明参数
error图片类头像加载失败的回调， 返回 false 会关闭组件默认的 fallback 行为(e: Event)

Slot #

事件名称description
default自定义头像展示内容
