Config Provider 全局配置 #

Config Provider 被用来提供全局的配置选项，让你的配置能够在全局都能够被访问到，Config Provider 使用了 Vue 的 provide/inject 特性

Config Provider 相关配置 #

通过 Config Provider 来配置多语言，让你的应用可以随时切换语言。
